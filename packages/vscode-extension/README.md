# Loi 25 Quebec — Snippets de consentement aux cookies / Cookie Consent Snippets

> **FR** | [English](#english)

Extraits de code pour ajouter rapidement le consentement **Loi 25 du Québec** à votre site web.

Supporte : **JavaScript**, **React (JSX/TSX)**, **HTML**, **PHP (WordPress)**

Par [Rayels Consulting](https://rayelsconsulting.com) — Agence numérique à Montréal

## Snippets

| Préfixe | Description | Langages |
|---------|-------------|----------|
| `loi25-banner` | Bannière de consentement aux cookies complète | JavaScript |
| `loi25-check` | Vérifier le consentement avant de charger les traceurs | JavaScript |
| `loi25-revoke` | Révoquer le consentement et recharger | JavaScript |
| `loi25-react` | Composant React de consentement complet | JSX / TSX |
| `loi25-hook` | Hook React pour le statut du consentement | JSX / TSX |
| `loi25-html` | Bannière via script HTML inline | HTML |
| `loi25-wp` | Extrait pour functions.php WordPress | PHP |

## Utilisation

1. Ouvrez un fichier supporté (JS, JSX, TSX, HTML, PHP)
2. Tapez un préfixe (ex. `loi25-react`)
3. Appuyez sur Tab pour développer
4. Remplissez les paramètres (langue, URL de confidentialité)

## Qu'est-ce que la Loi 25 ?

La Loi 25 du Québec exige que les sites web obtiennent un consentement explicite aux cookies de la part des utilisateurs. Ces snippets vous permettent d'ajouter des bannières de consentement conformes en quelques secondes.

# English

Code snippets for quickly adding **Quebec Law 25** cookie consent to your website.

Supports: **JavaScript**, **React (JSX/TSX)**, **HTML**, **PHP (WordPress)**

By [Rayels Consulting](https://rayelsconsulting.com) — Montreal Digital Agency

## Snippets

| Prefix | Description | Languages |
|--------|-------------|-----------|
| `loi25-banner` | Full cookie consent banner | JavaScript |
| `loi25-check` | Check consent before loading trackers | JavaScript |
| `loi25-revoke` | Revoke consent and reload | JavaScript |
| `loi25-react` | Full React consent component | JSX / TSX |
| `loi25-hook` | React hook for consent status | JSX / TSX |
| `loi25-html` | HTML inline script banner | HTML |
| `loi25-wp` | WordPress functions.php snippet | PHP |

## Usage

1. Open any supported file (JS, JSX, TSX, HTML, PHP)
2. Type a prefix (e.g., `loi25-react`)
3. Press Tab to expand
4. Fill in the placeholders (language, privacy URL)

## What is Loi 25?

Quebec's Law 25 requires websites to get explicit cookie consent from users. These snippets help you add compliant consent banners in seconds.
