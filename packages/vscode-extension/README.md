# Loi 25 Quebec Cookie Consent Snippets

Code snippets for quickly adding **Quebec Law 25** cookie consent to your website.

Supports: **JavaScript**, **React (JSX/TSX)**, **HTML**, **PHP (WordPress)**

## Snippets

| Prefix | Description | Languages |
|--------|-------------|-----------|
| `loi25-banner` | Full cookie consent banner | JavaScript |
| `loi25-check` | Check consent before loading trackers | JavaScript |
| `loi25-revoke` | Revoke consent and reload | JavaScript |
| `loi25-react` | Full React consent component | JSX / TSX |
| `loi25-hook` | React hook for consent status | JSX / TSX |
| `loi25-html` | HTML inline script banner | HTML |
| `loi25-wp` | WordPress functions.php snippet | PHP |

## Usage

1. Open any supported file (JS, JSX, TSX, HTML, PHP)
2. Type a prefix (e.g., `loi25-react`)
3. Press Tab to expand
4. Fill in the placeholders (language, privacy URL)

## What is Loi 25?

Quebec's Law 25 requires websites to get explicit cookie consent from users. These snippets help you add compliant consent banners in seconds.
