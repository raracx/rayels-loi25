04 — npm Package

@rayels/loi25-core

1.0.0 • Published by rayels

Lightweight cookie consent solution compliant with Quebec's Loi 25 (Bill 64). Bilingual (FR/EN), zero dependencies, ~2KB.

Quick Start

// Install
npm install @rayels/loi25-core

// Use via script tag
<script src="node_modules/@rayels/loi25-core/index.js"></script>
<script>
  Loi25.init({ lang: 'fr', theme: 'dark' });
</script>

API

import { init, getConsent, isAnalyticsAllowed } from '@rayels/loi25-core';

init({ lang: 'en', position: 'top' });

// Later: check before loading trackers
if (isAnalyticsAllowed()) {
  // load Google Analytics, Meta Pixel, etc.
}
